LinkVault Q8 iOS package

Configured bundle identifiers:
- App: com.linkvaultq8.app
- Share Extension: com.linkvaultq8.app.ShareExtension

Important fix in this package:
- The Share Extension product name is forced to LinkVaultShareExtension.appex.
- Stale/empty .appex references are cleaned from the Xcode project.
- App Group entitlements are disabled by default because the current Share Extension sends the link to the main app using linkvault:// and does not require App Groups. This avoids archive failures when App Store provisioning profiles do not include the App Groups capability.

If you later decide to store shared data directly between the extension and app, enable App Groups in Apple Developer for both identifiers and set USE_APP_GROUPS=true in Codemagic environment variables.
